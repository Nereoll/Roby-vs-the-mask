# Robby vs the Mask

## Description
 Robby vs the Mask is a simple clone of the classic Flappy Bird game implemented using the Pygame library in Python. Players control a robot and must navigate it through a series of giant masks without crashing into them. The game features simple graphics and a score counter to challenge yourself

## Usage
- Press the spacebar/up arrow key to make the robot use its jetpack fly upward.
- Avoid crashing into the big maks by flying through the gaps between them.
- The game ends when roby collides with a mask or the ground.

## Features
- Simple one-button control scheme.
- Randomly generated pipe obstacles to increase replayability.
- Score tracking to keep track of the player's progress.
